# Specification: Build Core Business Card Generator MVP

## Overview
This track covers the development of the Minimum Viable Product (MVP) for the Business Card Generator. The goal is to create a functional, single-page application that allows users to select a template, customize it with their professional info and logo, and export the result.

## Scope
- **Scaffolding:** Basic HTML5/CSS3 structure and project organization.
- **Template System:** At least 3 "Classic & Elegant" templates (Landscape/Portrait).
- **Editor:** Live text editing for Name, Title, Email, Phone, and Website.
- **Logo Support:** Client-side image upload and preview.
- **QR Code:** Integration of a vanilla JS QR code generator.
- **Export:** Implementation of PDF and PNG export using browser-native or lightweight vanilla JS libraries.

## Success Criteria
- User can successfully navigate from landing to a finished card.
- Live preview updates instantly with user input.
- Exported files (PDF/PNG) match the on-screen preview.
- Application runs with zero build steps and zero server-side dependencies.
- Code coverage for core logic is >80%.

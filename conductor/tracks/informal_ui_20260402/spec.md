# Track Specification: Informal/Whimsical UI Redesign

## 1. Overview
- **Goal:** Transform the current "bland and boring" Rockford interface into a whimsical, casual, and "home-grown" experience.
- **Problem Statement:** The current professional blue/gray aesthetic is too formal for some users and lacks character.
- **Success Criteria:** The interface should feel like a "scrapbook" or "personal project," while still maintaining full functionality and producing the same high-quality business card output.

## 2. Functional Requirements
- **Theme Overhaul:** 
  - Incorporate "Handwritten Fonts" for UI labels and buttons.
  - Replace standard UI icons with "Hand-drawn/Sketchy" versions.
  - Implement "Textured Backgrounds" (e.g., craft paper, cardboard, or felt) for the main application panels.
  - Add "Scrapbook" decorative elements like stickers, tape, or stamps.
- **Eclectic Color Palette:** Use a variety of vibrant and potentially mismatched colors ("Eclectic Scrapbook") to reinforce the informal feel.
- **Informal Layout:**
  - Use "Organic Shapes" (large, irregular border-radii) for panels and inputs.
  - Introduce "Tilted Elements" (slight rotations) for panels or decorative items.
  - Implement "3D Paper Layers" using layered drop shadows that mimic physical depth.
- **Core Functionality Preservation:** All existing features (template selection, logo upload, QR generation, export) must remain fully functional and easily accessible.

## 3. Non-Functional Requirements
- **Performance:** The introduction of textures and decorative elements should not significantly impact the application's loading speed or responsiveness.
- **Responsive Design:** The new informal layout must still be usable on mobile devices.
- **Accessibility:** Ensure that handwritten fonts remain legible and that color contrast meets basic accessibility standards despite the "eclectic" palette.

## 4. Acceptance Criteria
- [ ] UI labels use a handwritten/casual font.
- [ ] UI panels have textured backgrounds (e.g., paper/cardboard).
- [ ] UI panels use large, irregular border-radii and slight rotations.
- [ ] Panels appear as "layered paper" with appropriate depth/shadows.
- [ ] New "Eclectic Scrapbook" color palette is applied throughout.
- [ ] All original functionality (editing, preview, export) is verified to work correctly.
- [ ] Layout remains usable on mobile.

## 5. Out of Scope
- **Redesigning the Business Card Output:** The goal is to redesign the *application UI*, not the business cards themselves (though new informal templates could be a future track).
- **Complex Animations:** The user requested "Simple/Static" interactions.
